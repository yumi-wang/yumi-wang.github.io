CSS position values: Relative, Absolute, and Fixed

10 September 2022


What is the position property value?

The position property value is a positioning method for element layout. It is what you use to manipulate the position of the elements. There are 5 different types of position values:

  • Static
  • Relative
  • Absolute
  • Fixed
  • Sticky

Static

Before jumping into today's topic, we must introduce static, the default position. An element with static positioning is positioned according to the normal flow of the document. It is like a piece of paper that stays still and provides a platform for other elements to move on top of it. And so the top, right, bottom, left, and z-index properties does not affect it.


Relative

An element with this position value is placed in relation to its default position. It also follows the normal flow of the document. The top, right, bottom, and left properties are used to adjust the element away from its default position.

When an element is relatively positioned, its default space will remain and does not affect the position of any other elements. You can imagine there are three circles drawn on paper. We can cut one out and move it around on the paper; its default position (the hole) will still be there. The other circles will remain in their position and won't be affected.

Let's take a look at an example. The position of the egg is set to relative, and you can see its original position.

If you want an egg on toast, what we need to do is to move the egg to its left, away from its right (right: 100px;). The egg is on the toast, and its default position remains blank, not being taken by the pie.


Absolute

An element with this position value is removed from the document flow. It will be positioned relative to the closest positioned ancestor. If an element with absolute position has no positioned parent, it goes up to the grandparent element or the document body. And its default position will be taken by other element(s).

It's like working on a word document with three paragraphs on it. If you cut the first one, the second one will automatically go up to where the paragraph one was. However, in CSS positioning, instead of being gone, the element will appear on another layer where its closest positioned parent is.

Another example is we set the egg's position to absolute and adjust its position using top, bottom, left, and right. You can see the egg is taken out of the document flow, and its default position is taken by the pie.


Fixed

Fixed positioning is similar to absolute positioning. An element with this position value is removed from the document flow. And its default position will be taken by other element(s). Accept for an element is positioned relative to the viewpoint (the current window we are looking at) when its position is set to fixed. The fixed positioned element stays in the exact same place of your screen window even if you are scrolling. Just like the egg on the bottom-right of the window!
